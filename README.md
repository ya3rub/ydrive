# YDrive

a could storage app based on firebase built on top of React.js, Tailwindcss and DaisyUI.

### Setup

- clone the repo and `cd` into it.

- install deps
  
  ```
  yarn install
  ```

- setup firebase
  
  - setup a new project in firebase and fill in your credentials in `firebase.js` file

- launch !
  
  ```
  yarn dev
  ```
